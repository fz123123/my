## 操作步骤

### 通达信
1. 打开通达信软件
2. 输入股票代码（如：AAPL）
3. 查看K线图
4. 右键点击"分析报表" → "历史成交"
5. 导出数据或手动记录

### 同花顺远航版
1. 打开同花顺远航版
2. 输入股票代码
3. 查看K线图
4. 点击"数据" → "导出数据"
5. 选择CSV格式导出

### 大智慧365
1. 打开大智慧365
2. 输入股票代码
3. 查看分时图/K线图
4. 点击"工具" → "数据导出"
5. 导出历史行情数据

## 数据差异分析

### 常见差异原因

1. **复权方式不同**
   - 前复权：调整历史价格
   - 后复权：调整最新价格
   - 不复权：使用原始价格

2. **数据更新频率**
   - 盘中数据实时更新
   - 盘后数据可能有延迟

3. **成交量统计**
   - 部分软件不统计集合竞价成交量
   - 成交单位可能不同（手/股）

4. **时间格式**
   - 部分软件使用交易日时间
   - 部分软件使用自然日

## 注意事项

1. **选择相同的日期范围** - 确保对比的数据日期一致
2. **选择相同的复权方式** - 建议使用"不复权"进行对比
3. **注意涨跌停** - 涨跌停股票成交量可能为0
4. **检查数据完整性** - 确保没有遗漏的交易日
5. **记录数据来源** - 便于后续追溯问题

## 数据验证标准

如果差异超过以下阈值，需要进一步检查：

| 指标 | 最大允许差异 |
|------|------------|
| 收盘价 | ±0.5% |
| 成交量 | ±5% |
| 涨跌幅 | ±0.1% |
| 最高/最低价 | ±0.3% |
