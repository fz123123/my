## 如果您想安装桌面版

### 1. 下载并安装 Node.js
- 访问：[https://nodejs.org/](https://nodejs.org/)
- 下载 LTS 版本（目前是 v20.x）
- 双击安装包，一路点击 "Next" 完成安装
- **重启终端或重新打开 PowerShell**

### 2. 验证 Node.js 安装
打开新的 PowerShell 窗口，运行：
```powershell
node --version
npm --version
```

### 3. 安装 Claude Code
```powershell
npm install -g @anthropic-ai/claude-code
```

### 4. 运行 Claude Code
```powershell
cd c:\Users\Administrator\Documents\trae_projects\0
claude
```
